# 更新日志

## [1.0.0] - 2026-07-06

### 首个正式版本

- 所见即所得 Markdown 编辑（基于 Milkdown / ProseMirror）
- 源码编辑模式（基于 CodeMirror 6）
- 分栏视图（源码 + 实时预览，支持同步滚动）
- 多标签管理（固定、拖拽排序、批量关闭、重命名）
- 文件树侧边栏（打开工作区文件夹后浏览 .md 文件）
- 目录大纲（自动生成，点击跳转，滚动高亮）
- AI 写作助手（兼容 OpenAI 接口格式，快捷操作：润色、翻译、扩写、缩写、总结等）
- 会话持久化（自动恢复上次打开的标签、光标位置和界面状态）
- 导出 HTML / 纯文本 / PDF（通过浏览器打印）
- 搜索与替换（支持源码、预览、所见即所得三种模式）
- 快速切换（Ctrl+P 按文件名定位工作区文件）
- 明暗主题切换
- 专注模式 / 禅模式
- 拖拽打开 .md 文件或文件夹
- 自动保存（1 秒延迟）
- 最近文件和最近工作区记录
- 字数 / 字符数统计
- CommonMark 0.31.2 兼容性 99.4%

### 已知限制

- 当前仅支持 Windows 平台
- AI 为非流式响应
- PDF 导出依赖系统打印对话框
