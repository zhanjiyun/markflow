# 更新日志

## [1.0.2] — 当前开发版

### 新增
- 未命名标签内容自动落盘到应用数据目录，支持异常退出后恢复
- 外部文件变更检测：窗口获得焦点时自动检查文件修改时间
- 外部文件变更 toast 提示
- 文件树新建/重命名使用自定义对话框
- 窗口位置和大小跨会话记忆
- 文件关联启动：双击 .md 启动并自动打开目标文件
- 单实例文件转发：已有实例运行时，新文件自动转发到当前窗口

### 修复
- session.json 原子写入 + 备份恢复，防止写入中途崩溃导致数据丢失
- 关闭最后一个标签后回到主页（不再强制新建未命名.md）
- 窗口关闭按钮点击取消后再次点击无响应
- 打开文件时多余未命名.md 标签（占位标签自动替换）
- 应用最小化后双击文件窗口不回到前台
- 标签栏 UI 高度和间距优化
- 主页最近打开过多时滚动异常

### 工程
- markdown 测试配置与生产配置统一到共享模块
- 新增 38 项 Node.js 纯逻辑测试
- App.tsx 提取 useAppLifecycle hook
- Preview 代码复制按钮改为事件委托模式

## [1.0.0] — 2026-07-06

### 首个正式版本

- 所见即所得 Markdown 编辑（Milkdown / ProseMirror）
- 源码编辑模式（CodeMirror 6），支持语法高亮和格式化快捷键
- 分栏视图（源码 + 实时预览，同步滚动）
- 仅预览模式，支持字体缩放
- 多标签管理（固定、拖拽排序、批量关闭、重命名）
- 文件树侧边栏（打开工作区文件夹后浏览 .md 文件）
- 目录大纲（自动生成，点击跳转，滚动高亮）
- AI 写作助手（兼容 OpenAI 接口格式，快捷操作：润色、翻译、扩写、缩写、总结）
- 会话持久化（自动恢复打开标签、光标位置和界面状态）
- 导出 HTML / 纯文本 / PDF（通过浏览器打印）
- 搜索与替换（源码、预览、WYSIWYG 三种模式）
- 快速切换（Ctrl+P）
- 明暗主题切换
- 专注模式 / 禅模式
- 拖拽打开 .md 文件或文件夹
- 自动保存（1 秒延迟）
- 最近文件和最近工作区
- 字数 / 字符数统计
- 退出前未保存确认
- CommonMark 0.31.2 兼容性 647/651（99.4%）

### 已知限制
- 仅支持 Windows 平台
- AI 为非流式响应
- PDF 导出依赖系统打印对话框
