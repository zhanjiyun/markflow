## 3. 强调 (Emphasis)

### 3.1 斜体 \[ ]

这是 *斜体文字* 示例。
这也是 _斜体文字_ 示例。

### 3.2 粗体 \[ ]

这是 **粗体文字** 示例。
这也是 __粗体文字__ 示例。

### 3.3 粗斜体 \[ ]

这是 ***粗斜体文字*** 示例。

### 3.4 词内强调 \[ ]

foo*bar*baz  -- 词内的\*应该被识别
foo\_bar\_baz  -- 词内的\_应该被识别

### 3.5 非强调情况 \[ ]

foo \* bar \* baz  -- 空格分隔的不应该是斜体
foo\_bar\_baz  -- 下划线在词内不是强调


```javascript

};
```
