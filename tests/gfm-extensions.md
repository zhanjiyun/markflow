# GFM (GitHub Flavored Markdown) 扩展测试

> 测试 MarkFlow 对 GFM 扩展语法的支持

***

## 1. 表格 (Tables)

### 1.1 基本表格 \[ ]

| 姓名 | 年龄 | 城市 |
| -- | -- | -- |
| 张三 | 25 | 北京 |
| 李四 | 30 | 上海 |
| 王五 | 28 | 广州 |

### 1.2 对齐方式 \[ ]

| 左对齐 |  居中 | 右对齐 |
| :-- | :-: | --: |
| 内容  |  内容 |  内容 |
| a   |  b  |   c |

### 1.3 表格中的格式化 \[ ]

| 功能     | 语法           | 状态 |
| ------ | ------------ | -- |
| **粗体** | `**text**`   | ✅  |
| *斜体*   | `*text*`     | ✅  |
| `代码`   | `` `code` `` | ✅  |

### 1.4 空表格单元格 \[ ]

| 列A     | 列B     | 列C     |
| ------ | ------ | ------ |
| 有内容    | <br /> | 也有内容   |
| <br /> | 都有内容   | <br /> |

## 2. 任务列表 (Task Lists)

### 2.1 基本任务列表 \[ ]

* [x] 完成的任务

* [ ] 未完成的任务

* [x] 另一个完成的任务

### 2.2 嵌套任务列表 \[ ]

* [x] 主任务一
  * [x] 子任务 1.1

  * [ ] 子任务 1.2

* [ ] 主任务二
  * [ ] 子任务 2.1

***

## 3. 删除线 (Strikethrough)

### 3.1 基本删除线 \[ ]

这是 ~~被删除的文字~~ 示例。
正常文字 ~~划掉~~ 正常文字。

### 3.2 删除线中的其他格式 \[ ]

**~~粗体也被删除~~**
*~~斜体也被删除~~*

## 5. 脚注 (Footnotes)

### 5.1 基本脚注 \[ ]

这是一个带脚注的句子[^1]。

[^1]: 这是脚注的内容。

### 5.2 多个脚注 \[ ]

MarkFlow 是一个 Markdown 编辑器[^editor]。
它支持 WYSIWYG 和源码两种模式[^modes]。

[^editor]: Markdown Editor 的简称。

[^modes]: WYSIWYG 模式为默认模式，可通过 Ctrl+/ 切换到源码模式。
